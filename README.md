# Registration

This repository contains a React login/signup UI inside the [login-signup](login-signup) app folder.

## Project Layout

- [login-signup](login-signup): React app built with Create React App
- [login-signup/src](login-signup/src): Application source code
- [login-signup/src/Components/LoginSignup](login-signup/src/Components/LoginSignup): Login/signup form component and styles

## Quick Start

```bash
cd login-signup
npm install
npm start
```

## Build

```bash
cd login-signup
npm run build
```

## Deployment

The app is configured for Vercel from the [login-signup](login-signup) folder. Production builds are generated into [login-signup/build](login-signup/build).

If you redeploy, make sure the Vercel project root is set to [login-signup](login-signup) so it uses the correct build output.
